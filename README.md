Deploy-only repository for the public WizusLabs site.

- Live site: <https://wizuslabs.com/> (canonical, Cloudflare Pages)
- This hostname (<https://wizuslabs.github.io/>) remains a live secondary

Canonical app-policy URLs now live on <https://wizuslabs.com/>; legacy
`wizuslabs.github.io` policy URLs keep resolving for store listings and
binaries shipped before the 2026-06-10 domain migration.
